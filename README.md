# Voluki

A minimal personal blog built with Astro and Markdown.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Writing

Add posts in `src/content/blog`.

```md
---
title: "Post title"
description: "Short summary."
date: "2026-06-06"
tags: ["notes"]
---

Your post goes here.
```

## Cloudflare Pages

- Build command: `npm run build`
- Output directory: `dist`
